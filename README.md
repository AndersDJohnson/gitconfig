# gitconfig
My Git configurations.

## Install

#. Download
  * `git clone https://github.com/AndersDJohnson/gitconfig ~/.gitconfig-dir`

#. Run install script:
  * `cd ~/.gitconfig-dir`
  * `./install.sh`
